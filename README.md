# Uni-Plan
## Smart Course & Study Planner

**Uni-Plan** is a web application project aim to help college students organize and manage their academic responsibilities in one centralized platform.

Many College Students often manage multiple courses, assignments, and exams using different tools, which makes it difficult to see all tasks in one place, prioritize productivity, and stay organized

Uni-Plan solves this issue by providing a manageable web application that centralizes course information, assignment, exams and intelligent task prioritization to enhance productivity.

## Project Overview

Uni-Plan allows students to:

- Manage all courses in one place  
- Track assignments, quizzes, projects, and exams  
- View deadlines clearly  
- Receive prioritized task recommendations  
- Stay organized with a unified academic dashboard  

---

## Modules

Uni-Plan consists of several key modules:

### User Authentication & Profile Module
- User registration  
- Login and logout functionality  
- Basic profile management  

### Course Management Module
- Add courses  
- Edit course details  
- Delete courses  
- Store course-specific information  

### Assignment & Exam Management Module
- Create assignments, quizzes, projects, and exams  
- Set due dates  
- Track academic tasks  
- Monitor deadlines  

### Task Prioritization & Recommendation Module
- Implements a prioritization algorithm  
- Ranks tasks based on urgency and importance  
- Recommends what students should focus on first
